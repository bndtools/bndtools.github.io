The Bnd project has released **Bnd/Bndtools 7.3.0**, delivering a range of improvements across tooling, analysis capabilities, repository management, and developer experience. 

### Highlights

* **New AnalysisPlugin API** enabling additional build and analysis extensions.
* **Bundle Graph View** to help visualize bundle dependencies and relationships within OSGi applications. 
* **Declarative Services (DS) validation improvements** for earlier detection of component configuration issues. 
* **MavenBndRepository enhancements**, including support for using `pom.xml` files as repository indexes. 
* Numerous fixes, refinements, and ecosystem updates across the Bnd and Bndtools platform.
